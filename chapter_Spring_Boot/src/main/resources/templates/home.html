Данный проект создан для практики использования Spring Framework и прочих технологий, необходимых для создания приложений.

Основу приложения составляет Spring Boot. Оформление основано на фреймворке Bootstrap. Содержание и оформление подвергаются изменениям по мере добавления нового функционала.

Храниение информации осуществляется в БД Postgres. Взаимодействие с базой данных реализовано с использованием JDBC, JPA и возможностей Spring DATA.


Пункты меню указывают на следующие части функционала приложения:

  • Транзакция: использование Spring Transaction на примере перевода сумм между счетами аккаунтов. Ссылка на документацию
  • Безопасность: реализация Spring Security на основе доступа к страницам, требующим аутентификации или определенной роли пользователя. Страница регистрации демонстрирует валдицаю вводимых данных.
  • REST: Для CRUD операций используются стандартные методы HTTP: GET, POST, PUT, DELETE. Данные отправляются и получаются в форматах XML или JSON. Для конвертации форматов используется библиотека jackson-dataformat-xml. Взаимодействие с БД осуществляется посредством Spring Data JPA.
  • Файлы: Демонстрация загрузки файлов.
    • Загрузка одного или нескольких файлов стандартным способом.
    • REST загрузка по ajax с использованием jQuery.
  • Полезное: Прочие полезные примеры и особенности использования Spring
    • Использование аннотации @ConfigurationProperties для выбора варианта конфигурации.
